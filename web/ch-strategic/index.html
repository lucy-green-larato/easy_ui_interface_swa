Flexible matching: case-insensitive, OCR-typo tolerance, curated synonyms (e.g. global, overseas, worldwide). Single tag only.

2. Companies

Only Company Name and Company Number are read. Other columns are ignored.

Reuses the main report wiring to pull Company Name and Company Number.

3. Actions

Lists ≤ 50 run immediately. Larger lists (up to 5,000) run as a background job and stream matches into a downloadable CSV.
